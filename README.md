# Odin Recipes

This project is part of The Odin Project Foundations course.

The goal of this project is to build a basic recipe website using HTML. The website will contain a main index page with links to several recipe pages.

By completing this project, I will have demonstrated skills in:

- Creating basic HTML pages
- Working with links and images
- Using lists and text formatting
- Organizing files and folders correctly
- Using Git and GitHub for version control
- Cloning repositories and pushing changes

This project will help strengthen my understanding of the fundamentals of web development.
